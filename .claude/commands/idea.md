---
description: 记录一条随笔或灵感，追加到 ideas.md
argument-hint: <想法内容>
---

把用户这次输入的想法追加到项目根目录的 `ideas.md`。

**输入**：`$ARGUMENTS` 是用户这次要记录的想法原文，原样保留即可，不要改写、润色或扩写。

**步骤**：
1. 读取本地时间（UTC+8）的当前时间，格式为 `YYYY-MM-DD HH:MM`。如果不确定时间，用 `date "+%Y-%m-%d %H:%M"` 取。
2. 如果 `ideas.md` 不存在，新建它，首行写 `# Ideas`。
3. 以以下格式**追加**到文件末尾（注意前面留一个空行与上一条隔开）：

   ```
   ## YYYY-MM-DD HH:MM

   <用户原文>
   ```

4. 写完后只回一句简短确认，例如「已记录到 ideas.md (2026-04-27 22:13)」，不要复述内容、不要点评、不要给建议。

**约束**：
- 这是纯记录命令,不要联想到 memory 系统、不要触发摄入工作流、不要更新任何 finding/theme/principle。
- 不要给想法打标签或分类。
- 如果 `$ARGUMENTS` 为空,回复「想法呢？在 /idea 后面写下来」并停下,不要新建空条目。
