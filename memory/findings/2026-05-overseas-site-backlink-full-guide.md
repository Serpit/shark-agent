## 核心 SOP 或原理

### 一、第一性原理（决定所有策略的判断依据）

1. **外链=信任传递**：谷歌 PageRank 把链接当投票；权威站投票=高信任背书；无外链新站=零信任起步
2. **域名数 > 链接总数**：100 个不同域名各 1 条链接 ≫ 1 个域名给 1000 条链接（同一域名第二条起边际权重急剧衰减）
3. **三层排名逻辑**（哥飞）：title/h1 决定能否进待排名 → 外链决定能否进前 10 → 网站体验决定最终名次
4. **相关性 > 权威性**：DR40 行业网站 > DR90 不相关博客；DA85 但无关 < DA35 但精准相关
5. **质量 > 数量**：10 高质量胜 1000 垃圾；Fiverr $5/1000 链接基本全是机器垃圾，可能触发 SpamBrain 惩罚

### 二、六维好外链标准（droidHZ 综合版）

| 维度 | 判定 | 工具/操作 |
|---|---|---|
| **真实流量** | 有自然流量 + 品牌词搜索 | Semrush AS、AITDK |
| **网站活跃** | 近期有新页面被收录 | `site:domain.com` 选最近一周 |
| **DR/DA** | 越高越好但要交叉看 AS | Ahrefs SEO Toolbar 插件 |
| **运营时间** | 老域名 > 新域名 | AITDK 插件查注册时间 |
| **dofollow** | 占多数（不可能全部，那是异常信号） | 右键查源码看是否有 nofollow |
| **行业相关** | 与你网站类型相关 | 谷歌偏好相关性高的外链 |

**位置价值排序**：正文段落 > 专题页 > 侧边栏 > 页脚 > 评论区。正文嵌入效果是侧边栏/页脚的 5x+。导出链接 <10 个的外部页面是理想分母。

**锚文本建议比例**：品牌词 ~40% / 核心关键词 ~30% / 裸链 ~20% / 泛词 ~10%。全部精确匹配是 100% 红旗。

### 三、找外链 5 法

1. **Semrush 反向链接分析**：把对标域名扔进 Semrush → 看"最佳"外链 → 看"引荐域名" → 导出全部一条条筛
2. **Ahrefs Backlink Checker**：免费版只看 20 条，但通常都是高质量；连这 20 条都没拿下别想看更多
3. **谷歌搜索语法**（成本 0）：
   - `intext:你的域名 -site:你的域名` 找所有提及
   - `intext:你的域名 site:reddit.com` Reddit 提及
   - `intext:你的域名 site:producthunt.com` ProductHunt 提及
   - `intext:你的域名 site:news.ycombinator.com` HN 提及
   - 新站尤其好用（Semrush/Ahrefs 还没抓到时谷歌已有）
4. **外链聚合站**：mkdollar.com/backlinks、directories.bestaitools.com、egolinks.online/@backlinks、aimaker.dev/directory、sopilot.net/zh/submitdir、bestdirectories.org、directoryfame.com；用 Similarweb 看一个导航站的相关网站做"站找站"
5. **BuiltWith 追根溯源**：把"Disqus"/"Shopify"/"WordPress" 等丢进 builtwith.com → 找到所有用了该评论/UGC 系统的站点 → 评论区即外链入口

| 4 | **搬运工**（找改名/迁移/停服但还被链接的过期 URL → 联系站长替换为你的内容；不必断链 404，过时即可） | 免费 | 中 | 高 |

**鸭老师建议起步节奏**：基础外链 10-20 条/周。

### 五、付费外链——可以买曝光，不要买 PageRank

**Andy 600 美金实验筛选标准**（核心可移植）：
1. 月流量 > 50 万（AITDK 查）
2. Semrush AS 处没有 "Link Farm" 标记
3. **DR 和 AS 一起看**（DR 可虚高，AS 更真实）
4. 导航站要自带流量，不只是收录你的链接

最终：TAAFT（$347）+ Toolify（$99）效果最明显。

**七问筛选服务商清单**（鸭老师 SEO）：
1. 有样本吗？（Moz 查 DA + Spam Score）
2. 低垃圾分 + 高 DA？
3. 锚文本/URL/图片三形式都能做？
4. dofollow 还是 nofollow？（要 dofollow 占大头）
5. 保障多久？永久吗？
6. 丢失（404）免费补吗？
7. web2.0 还是 guest post？（防被偷换品类）

**底层原则**（大罗老师）：站内+内容先做好 → 重点攻 10-20 条高质量（不是 1000 条垃圾）→ 用免费方法做基础（HARO/资源页）→ 有利润再付费。**Andy 路径**：PMF 验证 → 有日收入 → 拿利润的一部分投外链 → 风险可控。

**10 平台速记**：Fiverr 试水筛选成本高 / Upwork 可定制 / Adsy guest post 透明 / LinksManagement 批量 / FatJoe 高质量站点池 / BackLinks 性价比 / SEOClerks 鱼龙混杂 / Sape.ru 俄语 / OutreachZ 博主外链 / BacklinksRocket 套餐型。

### 六、外链系统化策略

1. **70 内页 / 30 主页**——70% 外链指向博客/专题页（链接价值更大、出版商更愿接受、再内链传递回首页），30% 指向主页/频道页
2. **dofollow 占大部分但混 nofollow**（全 dofollow 反而异常）——nofollow 仍能带真实流量 + 品牌曝光 + Ahrefs 计算 DR 时也算入
3. **新站匀速增长**——一个月突增 100 条会被怀疑；从 10-20 条/周起步逐步扩大
4. **新站策略：外链放内页**——一旦某外链被污染，删那个内页即可，不拖累整站（很多人不知道的关键策略）
5. **锚文本多样化**——品牌/裸链/泛词/关键词自然混合，越不像"做"出来的越好

### 七、维护

- 每月用 Ahrefs/Semrush/GSC 检查丢失外链 → 联系站长恢复
- Disavow 慎用（谷歌 2016 后识别能力已强）：垃圾外链先尝试联系删除（留证据）→ 否则做 .txt → GSC 上传 → 每季度/半年复查
- 改版必设 301，否则历史外链全作废
- 常青内容定期更新（加新事实/更新统计数据）——比创新内容便宜且更能持续吸新链接

### 八、六大坑（每一个都是真金白银教训）

1. $5/1000 外链包（机器自动，关联博彩/成人）
2. AI 批量生成页面（用户停留时长极短 → 谷歌判低质 → 拖垮整站信任；老站救不回，建议换域名重起——刘屹案例）
3. 只看 DR 不看流量
4. 忽略相关性死磕 DA/DR
5. 质量没做好就猛发外链——"外链加速发现你的好页面"，前提是页面是好页面
6. Link Farm / 任何"100% 保证排名"承诺

## 风险提示

- **准公理 B 轻度冲突（窗口期）**：
  - 具体付费平台（TAAFT/Toolify 等）效果会随收录政策变化衰减；Andy 案例 2026-05 距今很近，但仍要复测当前 AS 数据
  - 平台规则演化：HARO 已并入 Featured.com（文中已反映）；Vercel 商用禁令（在百年 finding 已警告）；其他平台未来同样可能动政策
  - **降险动作**：建立"每季度复测一次主用渠道 DR/AS"的维护节奏；每次跑外链先用 `site:` 命令观察自家最近一周收录情况
- **公理 5 衍生（成本对齐）**：Guest Post $200-400/条 vs 当前支付通道约束——付费外链投入必须与 AdSense/订阅回流节奏匹配，PMF 未验证前不烧；优先把 5、7、8 中的免费高 ROI 打法（HARO / 资源页 / Notion / Google Sheet / 开源赞助 $5）跑透
- **公理 1 衍生（input 门槛）**：
  - Guest Post / 摩天大楼需要英文外联邮件能力（用户当前配合 AI 可达成，但语料库需要积累）
  - HARO/Featured.com 回复需要"专业非广告化"——非母语者要先把回复模板和领域表达练熟
  - BRD 阶段决定的"产品 Niche 是否容易找到相关高 DR 网站"是先决变量——窄到没相关站 = 没外链池
- **公理 4 衍生（流量到收入闭环）**：
  - Andy 流量翻倍 = 订阅 $40→$90/天，但前提是已经有付费产品；用户当前用 AdSense 路径，外链流量到收入的中间还隔着 AdSense 单价（小流量站极低）
  - 不要被"流量翻倍"叙事拉着投外链——先确认变现链路已通（即使是 AdSense 几美金也算通了）
- **公理 6 警惕（认知伪装拖延）**：把"我要先把外链学透"作为"还没准备好建站"的心理掩护是高发陷阱；正确顺序仍是百年 SOP——先建站、先内容、先选词，外链是 M2 才启动的环节
- **新风险模式（已回流 risks.md）**：
  - 批量垃圾外链 / Link Farm 服务
  - AI 批量生成页面拖垮整站信任
