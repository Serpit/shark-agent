## 第一层：普适商业规律（dbs 公理 1-6）

### 公理 1：模式独立于人——但 input 有门槛

**表述**：一个生意的成败首先取决于模式结构（供需关系、竞争格局、成本结构），其次才是执行者。但不意味着"谁做都一样"——input 侧（技术能力、行业知识、资源网络）构成准入门槛，模式好但 input 不匹配同样跑不通。

**触发判定**：
- 文章中只强调"谁都能做""零门槛"但回避 input 要求 → 大概率夸大
- 文章归因"成功全凭个人努力/聪明"但忽略模式结构优势 → 幸存者偏差信号
- 评估方向时，先问"这个模式的 input 要求我满足吗"，再问"这个模式本身好吗"

### 公理 2：模式决定道德

**表述**：在差模式里，好人也会做坏事。高压力/零和博弈/短期套利的商业模式会系统性扭曲参与者的行为。与其指责个体道德，不如审视模式是否给了作恶的激励。

**触发判定**：
- 文章鼓吹"利用信息差""趁平台还没管""抓紧窗口期收割" → 模式本身就奖励短期行为
- 看到一个方向的从业者普遍行为偏差（割韭菜、虚假宣传、刷量）→ 先怀疑模式而非人品
- 选择方向时：如果这个赛道"正经做事的人活不下去"，模式本身有问题

### 公理 3：智力不直接变现——区分认知与逃避

**表述**：底层认知（拆解能力、判断框架、风险识别）是商业成功的必要条件，但"想明白"离"赚到钱"中间隔着执行。过度分析、反复调研、追求"完全想通再做"是逃避不适感的伪装。

**触发判定**：
- 文章停留在"思维模型""底层逻辑""认知升级"但无任何操作步骤 → 认知贩卖，不产生行动价值
- 用户连续提出"再研究一下""还没完全想通"→ 用公理 6 心理卡点联动判定
- 摄入文章时区分：文章给的是"可操作的拆解"还是"不可证伪的认知框架"

### 公理 4：流量≠收入——多数情况不自动转化

**表述**：流量（粉丝数、播放量、PV、点赞）与收入是两回事。多数情况下，流量不会自动变成收入——需要定价、转化链路、变现设计。用流量数据替代收入数据做判断，会严重高估一个方向的商业价值。

**触发判定**：
- 文章用"涨粉 XX 万""播放 XX 万""月 PV XX"替代收入数据 → 公理 4 冲突
- 案例只披露流量指标但从未披露收入/利润/转化率 → 存疑，标注信息缺口
- 评估方向时：追问"流量到收入中间差哪几步""作者给没给真实收入数据"

### 公理 5：定价即产品——倍数按品类分

**表述**：价格不是在产品做完之后"加上去"的标签——定价本身就是产品的一部分，决定用户群、预期、体验和竞争位。**定价倍数参考**：知识付费/社群/课程类 5-15x 成本（高感知价值 + 非标品）；SaaS/工具类 2-5x 成本（有竞品对标 + 比价透明）。

**触发判定**：
- 文章中产品成本核算缺失但价格被包装成"超值" → 公理 5 冲突
- 独立开发者案例定价明显低于品类中位数（如 SaaS $3/月）→ 公理 5 低价陷阱
- 评估方向时追问："定价落在品类什么区间？是否匹配产品价值感知？"

### 公理 6：多数卡点是心理问题——经验上约 2/3 起

**表述**：多数创业卡点（经验上约 2/3 起）不是信息不足或能力不够，而是心理阻力——害怕失败、害怕被评判、完美主义、损失厌恶伪装成"理性分析"。区分方式：如果已经给了足够信息但迟迟不行动，且用户给出的理由循环回到"还没准备好/再看看"，就是心理问题。

**触发判定**：
- 用户出现信号词："我担心""会不会失败""还没准备好""再调研一下""等 XX 之后" → 当轮提醒
- 同话题连续 ≥2 轮回到"还没决定/再想想" → 追问"这是信息问题还是心理问题？"
- 用户明确说"我知道是心理问题"或"先放着" → 标记已识别，本会话不再重复
- 同话题触发后冷却 3 轮

## 第二层：出海/副业应用启发式（shark 准公理 A/B/C）

普适性低于第一层，未来可能随实战扩充或淘汰。扫描时判定权重：dbs 公理冲突 → 大概率 anti-pattern；准公理冲突 → 打问号追问。

### 准公理 A：形态决定复利——副业必须选"人不在也跑"的形态

**表述**：副业的商业价值由其形态的复利特征决定。资产型（SEO 站/内容矩阵/工具订阅/模板）> 卖时间型（定制/咨询/辅导）。判断标准：人不在收入是否继续。

**触发判定**：
- 评估机会时先问："这件事人不在还跑不跑？"
- 若停了就停了 → 警惕它对其他复利线的时间挤压
- 既是"做什么"的判断维度也是"不做什么"的排除标准

### 准公理 B：窗口期优先于复制案例——所有"X 个月赚 Y"先问窗口期

**表述**：任何声称"X 个月赚 Y"的案例，判定层级的第一问是：**案例发生的窗口期现在还在吗？** 平台规则、竞争密度、算法红利、入场门槛随时间快速变化——复制方法但错过窗口，结果可能差 10 倍。

**触发判定**：
- 案例距今 >12 个月 → 窗口期大概率已关闭，不能按原文收益假设复制
- 案例距今 6-12 个月 → 需要交叉验证当前从业者的最新数据
- 案例距今 <6 个月 → 可能存在局部窗口，但警惕作者是否有动机夸大（卖课/引流）
- 文章出现"现在还来得及""红利还在"但未给出当前数据证明 → 准公理 B 冲突

### 准公理 C：模式 vs 努力的分层判断——出现"我比别人努力 X 倍"先问模式天花板

**表述**：当用户或文章把成败归因到"比别人更努力"时，先往上追一层：这个模式的**努力-产出函数**是什么形状。如果模式本身天花板由"小时数 × 单价"决定（卖时间型），加倍努力只是线性增长；如果模式有经营杠杆（资产型），努力才可能带来非线性回报。

**触发判定**：
- 文章暗示"只要够拼命就能成功" → 追问模式天花板
- 用户出现"我比别人努力但没结果" → 先排查是否在差模式里硬卷
- 与准公理 A 联动：形态不行 → 努力只能线性放大 → 按复利标准重新选形态
