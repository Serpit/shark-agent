# 时间规划

## 阶段总览

- **2026-04 至 2026-07 上旬**:知识收集与沉淀阶段——目标是锁定出海方向、挖出具体市场。这个阶段优先广度与质量分类,不急于行动。
- **2026-07 起**:小规模试错阶段——从这之后,agent 的输出应更偏向「这个 SOP 我能不能马上跑一个最小验证」而非继续囤积信息。

## 双轨节奏(IP 优先,创业探索并行)

总盘:18 小时/周(工作日 2h × 5 + 周末 8h)。详细执行规则见 [experiments.md](experiments.md) 中的「个人 IP 定位与内容主线验证」实验。

### M1(2026-05):IP 起势 + 建站能力前置

- **创业探索(主线变更:SEO 内容站)**:跑通建站完整流程——Astro + Tailwind + Cloudflare Pages 模板上线 1 个站;用 [search-engine-demand-discovery.md](methods/search-engine-demand-discovery.md) SOP 选 1 个 KD<20 + 月搜索量 500-2K 的关键词;首版内容 5-10 篇 AI 生成 + 人工校验
- **创业探索(并行轻量)**:跑 SEO 站过程中观察是否有 transactional intent 关键词背后是 SaaS 需求,记录到 [experiments.md SaaS 候选池](experiments.md#技术型出海小工具候选池调研web-first) 但不启动 BRD
- **IP**:X 累计 ≥12 条小推文 + ≥1 篇长文/总结;公众号 1-2 篇;小红书随手发。**内容素材自然切到"建站实战"**——选词、上线、踩坑都是真实素材
- **不启动**:YouTube、SaaS BRD、Stripe 主体研究
- **月底产出**:1 个上线 SEO 站(域名 + 5-10 篇内容 + 已提交 Google Search Console)+ 一个能解释清楚的 X 定位

### M2(2026-06):内容产能 + 外链系统启动

- **创业探索**:站点扩到 3-5 个;启动百年的"竞品外链 + 半自动化提交"流程(吸收 Claude Code Skill / Agent / Hook 工程范式);AdSense 申请提交(需站点有持续内容才易过)
- **IP**:X 内容切到"SEO 实战 + 外链系统搭建"叙事;月末启动 YouTube 第一支(用 SEO 实战做素材)
- **公众号**:产出 1 篇 SEO 站从 0 到 1 全流程长文
- **月底产出**:3-5 站上线 + AdSense 已申请(未必通过)+ 第一波关键词进入 Google Top 100

### M3(2026-07):矩阵铺开 + 第一笔 AdSense

- **创业探索**:站点扩到 5-10 个(参考百年 4 个月 20-30 站,我们是兼职节奏减半);DR 目标突破 10-20;**第一笔 AdSense 入账即视为 M3 GO**(金额几美金即可)
- **IP**:进入实战复盘模式;评估是否带来 ≥3 条同频连接 / 行业线索
- **SaaS 路径决策**:此时回头评估——SEO 流量起来后是否有 1-2 个关键词背后能升级为 SaaS?同时调研 Lemon Squeezy / Paddle 主体方案,为 M4+ 做准备
- **节奏切换判定**:若 SEO 流程跑通 + 第一笔 AdSense + IP 已成型 → 进入"试错放大"阶段(扩矩阵 + SaaS BRD 启动)。若三者都没,触发下方降级路径

## 节奏提醒触发条件

- **M1 中段(每两周)**:若 X 周更新 < 5 条或连续 2 天断更 → 触发 IP 实验复盘,先砍小红书/公众号,保 X
- **M1 末**:若 X 累计 < 12 条 → 说明 18h 时间硬约束严重高估,要求重新校准 [profile.md](profile.md#硬约束)
- **M1 末**:若 SEO 第一站没上线 → **不是延期问题,是建站流程门槛超出 18h/周可承担范围**;考虑用更轻的载体(Carrd / Notion 公开页)先跑流程,再迁 Astro
- **M2 末**:若 3-5 站没铺出来 → 调研是否陷入"完美主义建站"陷阱,违反 [principles.md#先发再迭代](principles.md#先发再迭代-做出-v01-优于完美计划)
- **临近 2026-07**:若 0 个站进 Google Top 100 → 不是策略错,是关键词选错;重跑 [search-engine-demand-discovery.md](methods/search-engine-demand-discovery.md) Step 2 收紧 KD/搜索量门槛
- **进入 2026-07 之后**:若用户开始焦虑"AdSense 才几美金" → 主动校准期望:百年 20-30 站才月几十美金,这是 9-18 月节奏,不是 M3 节奏

## 期望校准(关键)

**M1-M3 不追求收入指标**,主线 KPI 是:
- M1:**1 站上线 + 流程跑通**
- M2:**3-5 站铺开 + AdSense 申请**
- M3:**5-10 站矩阵 + DR>10 + 第一笔 AdSense(几美金即 GO)**

收入跃迁(月入百美金量级)预期 M6-M9,不是 M3。**这条期望线一旦模糊,就会陷入"努力但达不到"心理负反馈**。
