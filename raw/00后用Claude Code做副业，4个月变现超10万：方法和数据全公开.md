今年1月开始认真用 Claude Code 做副业。4个月后回头一算：抖音接单7万，SEO站二三十个，自动化工具也卖了几千块，站点广告收入开始奔跑，还有教培的收入。加起来超过10万。

这不是一个"AI一键赚钱"的故事。中间有大量试错、沟通、返工和脏活累活。只是 Claude Code 把一个人的交付上限拉高了很多，让我第一次感觉到：副业不一定只能靠卖时间，也可以靠系统、工具和资产慢慢滚起来。

这篇文章把我的背景、具体做法、工具选择和真实数据都摊开讲。

# 关于我

我是百年，00后，NUS 数据科学毕业，现在主业做跨境电商。

在做 AI 副业之前，我做了四五年新加坡留学考试辅导。这个赛道很窄，需求稳定，生源也相对固定。每年能有三四万收入，对学生也确实有帮助，但它有一个很明显的问题：我辅导一个学生，就只能赚一个学生的钱。停下来，收入也就停了。

去年底，我开始认真思考一件事：有没有一种副业，不是每次都从零开始服务一个客户，而是做完之后还能继续产生价值？

也是这个时候，我把主要精力放到了 Claude Code 上。

AI工具我用得比较早。2023年底开始付费用 ChatGPT，2024年切到 Cursor，2025年4月开始重度使用 Claude Code。GPT、Claude、Gemini、Cursor 这些工具我都完整用过一轮，最后留下来的标准很简单：哪个能真实提高交付效率，就用哪个。

到现在，我比较自豪的一点是，单个 200 美金的 Claude Code Max Plan 已经不够我用了。我目前是两个 Max Plan 同时跑，基本把它当成一个长期合作的工程团队在用。

# Claude Code 真正改变的，是一个人的交付半径

很多人理解 AI 编程，还停留在"帮我补全一行代码"。

但 Claude Code 对我来说不是补全工具，而是工程协作工具。它可以通读整个项目，理解几十个文件之间的依赖关系，然后完成一组完整任务：改后端接口、接前端页面、补数据库字段、改 Docker 配置、顺手跑测试。

为了把效率吃满，我给自己搭了一套固定工作流：

- **Skill**：把常用能力沉淀成可复用模块，比如部署、SEO审计、站点初始化。
- **Agent**：把多步骤任务拆成自动化流程，比如建站、抓取、清洗、发布。
- **Hook**：在提交代码后自动跑格式化、检查和基础安全扫描。
- **凭证管理**：所有项目按名称分文件夹管理，部署脚本自动读取环境变量。

![Image](https://pbs.twimg.com/media/HG7E7LMa0AAqbos?format=jpg&name=large)

这套系统跑顺以后，我的产能变化很明显。

一套完整的数据看板，后端 FastAPI，前端 Next.js，数据库 PostgreSQL，再加 Docker Compose，两三天可以交付一个可用版本。一个自动化工具，一周能跑起来。一个 SEO 站点，从选词、建站到上线，快的时候一个下午就能完成。

这也是我后来能同时推进抖音获客、企业项目、SEO站群和工具产品的底层原因。

不是我一个人突然变成了全栈团队，而是很多过去需要团队协作的工程步骤，现在被我拆成了可以让 AI 稳定执行的流程。

# 抖音技术图文：9篇内容，接到一个7万多的企业项目

2026年1月，我开了一个抖音号，只发图文，不做视频。

当时的判断很简单：技术类内容在抖音上大多是视频，图文反而很空。对我来说，图文成本也低。一篇内容从截图、排版到发布，大概一两个小时。

我发的不是教程合集，也不是标题党。基本就是我用 Claude Code 做项目的真实过程：今天在搭什么系统，遇到了什么问题，怎么拆任务，最后跑出来什么效果。截图加说明，信息密度高一点就够了。这里吃到了一个红利，抖音当时强推图文，我的抖音作为一个冷启动账号，9篇几乎全都过万，有三篇是10万+，涨粉速度不低，大概一周左右达成千粉，最终是不到两千粉。

![Image](https://pbs.twimg.com/media/HG7FJIwbwAA4-M8?format=jpg&name=large)

粉丝只有几百的时候，企业客资就开始进来。前后一共有七八个企业主动私信问合作。大部分需求其实并不匹配。这个也很正常，抖音流量大，但人群杂，而且比较下沉。最后真正成交的是一个和我的背景比较匹配的项目：一家海外企业要做多源数据聚合的实时看板，涉及几十个异构数据源、实时采集、地图可视化，以及 AI 自动生成分析简报。

这个项目最终签的合同是7万多。

技术上，它是一个典型的全栈数据工程项目：后端负责数据采集和 API，前端负责可视化仪表盘，数据库存储时序数据，地图和图表展示关键指标，再接大模型生成自动分析。整个系统通过 Docker Compose 一键启动，最后部署在一台服务器上。整体需要接AI，并且长期记忆化，用到了openclaw的一些思想。

从启动到 MVP 交付，用了不到一周。但更难的是全量交付，整体你在完整构建一个真实的，企业级别的，大型的系统。项目到现在还在持续交付，客户需求会变，沟通成本也不低。按小时折算下来，7万并没有听起来那么轻松。这个活儿目前我还在干，预估还需要做一两个月吧。

它真正验证的是另一件事：市场愿意为"用 AI 高效交付复杂项目"的能力付费。

后来我停更了抖音。不是平台不行，而是账号定位不够精准。来的客资里，真正匹配的太少，而且消磨灵魂。几十个客资里面真正能做的就是这个单子而已，再者我不想继续做非标项目，定制化的生意。

我更想做有复利的东西。

# SEO出海：批量建站，开始做能自动运转的资产

SEO出海我去年就接触过，但真正系统投入，是今年开始。

它和短视频完全不一样。短视频的逻辑是持续发布，持续获得推荐；SEO的逻辑是前期把页面、内容和外链做好，排名起来以后，流量会持续进来。

这更接近我想要的副业形态：不是一次性交付，而是长期资产。

这段时间，我一共上线了二三十个站。

![Image](https://pbs.twimg.com/media/HG7FejgbMAA_QPV?format=jpg&name=large)

我的流程是这样的：

**选词**：先用 Ahrefs 筛选有搜索量、竞争度低的关键词。核心原则是先确认需求，再建站。最近才出现搜索量的新词，竞争通常最小，也更容易起排名。

**建站**：框架用 Astro，样式用 Tailwind CSS，部署到 Cloudflare Pages。模板跑顺以后，单个站点两三个小时可以从零到上线。

**内容**：AI 负责生成初版，我负责人工校验。这里我不会完全放手，因为 SEO 站点如果内容不准，后面流量起来也留不住。

**内页与外链**：持续扩充长尾词页面，同时补充高质量外链，提升站点权重。

**变现**：目前主要通过 Google AdSense，手动配置几个核心广告位，再用自动广告做补充。

现在这批站的广告收入还不大，每月几十美金。但它给我的感觉和教培完全不同：教培是我今天不上课，今天就没有收入；SEO站点是我睡觉的时候，它也可能产生点击和收入。

金额还小，但是这是长期复利，是真正的资产。

# 半自动化外链系统：真正值钱的是流程库

SEO里最耗时间的环节是外链。

手动做外链非常繁琐：找目标站点、注册账号、填写资料、提交链接、验证是否收录。每个平台的规则都不一样，今天能跑通的方法，明天可能就失效。

我手动做了几天后，就用 Claude Code 和 Playwright MCP 做了一套半自动化提交系统。

这套系统最重要的不是脚本，而是 PostgreSQL 里沉淀下来的平台流程库。核心表记录了几千个平台的信息：平台类型、提交入口、注册方式、审核规则、注意事项、成功率、失败原因。

![Image](https://pbs.twimg.com/media/HG7FkpJaYAA_9u8?format=jpg&name=large)

完整流程大概是：

1. 从 Ahrefs 导出竞品外链数据
2. 导入系统，过滤 spam、低质量站点和已提交记录
3. 用 AI 辅助判断目标域名质量
4. 人工确认一批可做目标
5. Playwright MCP 半自动执行注册、填写和提交
6. 提交完成后，系统自动回查外链是否生效

我也试过全自动，结论是不现实。每个站点的流程差异太大，纯脚本覆盖不了。真正稳定的方式，是让 AI 做判断和执行辅助，人来把关关键节点。

竞品外链是成功率最高的来源。我的做法是找同赛道近两年起来的竞品，导出它们排名靠前的外链，再判断哪些可以复制。博客评论这类我也试过，成功率低、spam 比例高，后来基本放弃。

目前整体提交成功率大概在30%左右。按10个站点一批，通常有三四个能成功。

两周实测下来，6个新站提交了300多条外链。几个站的 DR 都从0开始明显提升，其中有站点突破了40，最高的超过50。

这件事给我最大的启发是：很多人以为工具的价值在代码，但真正值钱的是流程数据。代码可以重写，平台经验和失败记录很难凭空生成，其实完全类似于为什么工业软件值钱的逻辑。

# Skill 包变现：把重复劳动打包成产品

后来我在社群里分享这套外链自动化方法，有人直接问能不能买。

我就把方法论、Skill 包和平台流程库打包卖了一版，变现了几千块。

这套产品的核心价值在于节省时间，买的人不需要再自己一个站一个站测试，不需要重复踩注册、提交、审核这些坑。

这也让我重新理解了产品化：不一定非要做一个 SaaS。只要你把一套高频、痛苦、可复用的流程整理出来，它就可能成为一个小产品。并且，eat your own dog food 确实是最好的产品 idea 模式，因为你就在解决你自己的真实的痛点，你也完全清楚它的卖点。

# 4个月完整账目

![Image](https://pbs.twimg.com/media/HG7IIKtaQAAfBkh?format=jpg&name=large)

这4个月的收入主要来自三部分：

- 企业定制项目：7万多
- 教培收入：3万左右
- Skill 包和方法论产品：几千块
- SEO 站点广告收入：每月几十美金，仍在增长

如果只看金额，企业项目是大头。但如果看长期价值，我更看重后两件事。

企业项目证明了交付能力；Skill 包证明了经验可以产品化；SEO站点证明了资产可以自动产生现金流。

这三个方向加在一起，才是这4个月真正有意义的地方。
